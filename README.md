# My Quran - Private Quran Reader

A minimalist, luxury-themed Quran web application for personal use.

## Attribution

Quran data provided by [Quran JSON](https://github.com/rioastamal/quran-json?tab=MIT-1-ov-file) by Rio Astamal.

## Features

- **Offline First**: Uses local JSON data.
- **Luxury Design**: Minimalist aesthetic with premium typography.
- **Mobile Optimized**: Smooth performance on Android devices.
